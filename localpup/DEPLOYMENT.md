## 🔧 技术架构

### 前端技术栈
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **状态管理**: React Hooks
- **国际化**: 自定义i18n系统

### 数据架构
- **酒店数据**: 本地TypeScript数据文件
- **图片资源**: Unsplash CDN
- **配置管理**: 环境变量 + JSON配置

### 部署架构
- **托管平台**: Vercel
- **CDN**: Vercel Edge Network
- **域名**: Vercel托管域名
- **监控**: Vercel Analytics

## 🚀 部署流程

### 自动部署
1. 推送代码到GitHub main分支
2. Vercel自动检测并构建
3. 自动部署到生产环境
4. 域名自动更新

### 手动部署
```bash
# 1. 提交代码
git add .
git commit -m "更新描述"
git push origin main

# 2. 检查构建 (可选)
npm run build

# 3. 访问线上地址
open https://www.localpup.online
```
